# Carousel Item

A slide component that contains content you wish to display (e.g., image, text, etc.). Must be nested inside a `Carousel` component.

## Carousel Item Interval Prop

The Carousel.Item `interval` prop determines the amount of time to delay between automatically cycling a specific item. Will default to the Carousel's `interval` prop value if none is specified.
