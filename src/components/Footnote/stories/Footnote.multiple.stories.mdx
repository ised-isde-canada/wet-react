# Multiple

## Multiple Sub-Links

Footnote links that go to the same Footnote must differentiate themselves using the `subValue` attribute, so the Footnote will know which instance of the link to return to

To apply, simply give each FootnoteLink of a given `value` its own custom `subValue`

## Multiple Links

Text that links to multiple footnotes can simply be given multiple `<FootnoteLink value="..." />` blocks

## Multi-Referenced Footnote

Footnotes that are referenced multiple times are handled automatically, and will by default return to the first link that goes to them

`<FootnoteSection.Footnote value="..." children={...}/>`

## Multi-Paragraph Footnote

Footnotes containing two or more paragraphs are supported. They can take the form of either a standard or multi-referenced footnote.

`<FootnoteSection.Footnote value="..." children={<span><p>Par1</p><p>Par2</p></span>}/>`

<Canvas>
