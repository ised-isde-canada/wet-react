# Forms

## Form Check

For the non-textual checkbox and radio controls, FormCheck provides a single component for both types that adds some additional styling and improved layout.

### Name

You can give checkboxes a `name` attribute to group them by name. Use one name for every group of buttons

**IMPORTANT:** You **MUST** give a `name` attribute to radio buttons, otherwise checking one won't uncheck the others

Note: radio buttons only group by name with other radio buttons in the same `<form>` block

### Required

You can set a checkbox or radio as required using the `isRequired` prop.

By default it will show the "required" styling but you can set the visibility using `showRequiredStyling`

You can also customize the required message using the `requiredText` prop

<Canvas>
  <Story name="Required">
    <Form>
      <Form.Check
        type="checkbox"
        label="required field with visible styling"
        name="group4"
        isRequired
      />
      <Form.Check
        showRequiredStyling={true}
        isRequired
        type="checkbox"
        label="required field with visible styling"
        name="group4"
        requiredText="custom message"
      />
      <Form.Check
        showRequiredStyling={false}
        isRequired
        type="checkbox"
        label="required field with hidden styling"
        name="group4"
      />
      <Form.Check
        isRequired
        type="radio"
        label={`radios do not show styling`}
        name="group4"
      />
    </Form>
  </Story>
</Canvas>

### Invalid

You can set a checkbox or radio as "invalid" using the `isInvalid` prop.
