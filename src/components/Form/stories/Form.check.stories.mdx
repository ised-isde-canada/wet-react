# Forms

## Form Check

For the non-textual checkbox and radio controls, FormCheck provides a single component for both types that adds some additional styling and improved layout.

### Default

<Canvas>
  <Story name="Default">
    <Form>
      <Form.Check
        type="checkbox"
        label={`default checkbox 1`}
        value="checkbox-1"
      />
      <Form.Check
        type="checkbox"
        label={`default checkbox 2`}
        value="checkbox-1"
      />
    </Form>
    <Form>
      <Form.Check type="radio" label={`default radio 1`} value="radio-1" />
      <Form.Check type="radio" label={`default radio 2`} value="radio-1" />
    </Form>
  </Story>
</Canvas>

### Disabled

You can choose to disable a checkbox or a radio controls

<Canvas>
  <Story name="Disabled">
    <Form>
      <Form.Check type="checkbox" label={`default checkbox 1`} />
      <Form.Check isDisabled type="checkbox" label={`disabled checkbox 2`} />
    </Form>
    <Form>
      <Form.Check type="radio" label={`default radio 1`} />
      <Form.Check isDisabled type="radio" label={`disabled default radio 2`} />
    </Form>
  </Story>
</Canvas>

### Required

You can set a checkbox or radio as required using the `isRequired` prop.

By default it will show the "required" styling but you can set the visibility using `showRequiredStyling`

You can also customize the required message using the `requiredText` prop

<Canvas>
  <Story name="Required">
    <Form>
      <Form.Check
        type="checkbox"
        label="required field with visible styling"
        isRequired
      />
      <Form.Check
        showRequiredStyling={true}
        isRequired
        type="checkbox"
        label="required field with visible styling"
        requiredText="custom message"
      />
      <Form.Check
        showRequiredStyling={false}
        isRequired
        type="checkbox"
        label="required field with hidden styling"
      />
      <Form.Check
        isRequired
        type="radio"
        label={`radios do not show styling`}
      />
    </Form>
  </Story>
</Canvas>

### Invalid

You can set a checkbox or radio as "invalid" using the `isInvalid` prop.

<Canvas>
  <Story name="Invalid">
    <Form>
      <Form.Group isInvalid>
        <Form.Check type="checkbox" label="invalid checkbox" />
        <Form.Check type="radio" label="invalid radio" />
      </Form.Group>
    </Form>
    <Form>
      <Form.Check type="checkbox" label="invalid checkbox" isInvalid />
      <Form.Check type="radio" label="invalid radio" isInvalid />
    </Form>
  </Story>
</Canvas>

### Inline

Group checkboxes or radios on the same horizontal row by adding the `isInline` prop.

<Canvas>
  <Story name="Inline" style={{ width: '500px' }}>

      <Form.Check isInline type="checkbox" label={`checkbox 2`} />
      <Form.Check
        isInline
        isDisabled
        type="checkbox"
        label={`disabled checkbox 2`}
      />
    </Form>
    <Form>
      <Form.Check isInline type="radio" label={`default radio 1`} />
      <Form.Check
        isInline
        isDisabled
        type="radio"
        label={`disabled default radio 3`}
      />
    </Form>
  </Story>
</Canvas>
