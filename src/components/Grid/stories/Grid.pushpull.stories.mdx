# Grid Width

One row has a width of 12. Use the `xsPush`, `smPush`, `mdPush`, `lgPush`, `xsPull`, `smPull`, `mdPull`, `lgPull` attributes to specify
the pull and push (out of 12) that a given cell will have on a given viewport size (xs, sm, md, lg). This allows you to swap cell positions
on different viewport sizes

## example

<Canvas>

    </Grid.Cell>
    <Grid.Cell mdWidth={3} mdPull={9}>

    </Grid.Cell>
  </Grid.Row>
