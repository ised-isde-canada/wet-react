# Lightbox

## Gallery

Use the `Lightbox.Gallery` element to make a gallery for displaying multiple lightboxes

## Example

<Canvas>
  <Story name="Default" args={{}}>
    {Template.bind({})}
  </Story>
</Canvas>

## Tag

Use the `tag` attribute to set the HTML element that will act as the gallery (default is `'section'`)

For example: an unordered list `ul` of lightboxes

## Hidden

To hide lightboxes in a gallery, for example all but the first lightbox, use the `hidden` attribute in the
undesired lightboxes

There is also a `hide` attribute in gallery, which hides all but the first lightbox, but it only takes effect once a lightbox is clicked

## Example with hidden

<Canvas>
  <Story name="Hidden" args={{ hidden: true }}>
    {TemplateTwo.bind({})}
  </Story>
</Canvas>
