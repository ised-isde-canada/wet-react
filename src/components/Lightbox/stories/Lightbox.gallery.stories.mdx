# Lightbox

## Gallery

Use the `Lightbox.Gallery` element to make a gallery for displaying multiple lightboxes

Make sure to set the `gallery` attribute of each lightbox in the gallery to true

## Example

<Canvas>
  <Story name="Default" args={{}}>
    {Template.bind({})}
  </Story>
</Canvas>

## Tag

Use the `tag` attribute to set the HTML element that will act as the gallery (default is `'section'`)

## Example

<Canvas>
  <Story name="Tag" args={{ tag: 'ul' }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Hide

Use the `hide` attribute to hide all but the first lightbox in the gallery

## Example

<Canvas>
  <Story name="Hide" args={{ hide }}>
    {Template.bind({})}
  </Story>
</Canvas>
