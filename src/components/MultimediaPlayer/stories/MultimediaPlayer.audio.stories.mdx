# Multimedia Player

## Audio

Embed an audio element by adding a single source element with `type: mp3` or `type: ogg`<br/>

## Example

<Canvas>
  <MultimediaPlayer
    sources={[
      {
        type: 'mp3',
        source: valkyries,
      },
    ]}
    figCaption={<p>Ride of the Valkyries</p>}
    shareURL="https://www.archive.org/download/RideOfTheValkyries/ride_of_the_valkyries_2.mp3"
  />
</Canvas>
