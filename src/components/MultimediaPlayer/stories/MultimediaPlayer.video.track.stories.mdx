# MultimediaPlayer

## Track

Add a track element by setting <br/>
`trackSrcLang` to a shortform (2-character) language indentifier, such as `en` or `fr`<br/>
`trackLabel` to some shortform descriptive text for displaying to the user<br/>
`trackSrc` to the URL leading to the captions track<br/>
`trackDataType` to the type of captions provided, either `text/html` or `application/ttml+xml`
