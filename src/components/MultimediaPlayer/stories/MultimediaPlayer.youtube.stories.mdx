# MultimediaPlayer

using the `size` prop it is possible to set the size of the MultimediaPlayer to large `lg` or small `sm`

## Default

Use for typical MultimediaPlayer

<Canvas>
  <Story name="Default" args={{ children: '' }}>
    {MultimediaPlayerTemplate.bind({})}
  </Story>
</Canvas>

## Large

Use when the MultimediaPlayer needs additional prominence on the page. <br/>Since the default buttons already create prominence and a call to action, there is limited value/need to increasing their size

<Canvas>
  <Story name="Large" args={{ children: '', size: 'lg' }}>
    {MultimediaPlayerTemplate.bind({})}
  </Story>
</Canvas>

## Small

Use when the MultimediaPlayer appears in a small section of space. This reduces the button size to maximize the amount of buttons that appear in a single row

<Canvas>
  <Story name="Small" args={{ children: '', size: 'sm' }}>
    {MultimediaPlayerTemplate.bind({})}
  </Story>
</Canvas>
