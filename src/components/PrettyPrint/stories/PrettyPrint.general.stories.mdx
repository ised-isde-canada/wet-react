# PrettyPrint

## Initialize

Use the `initialize` attribute to initialize the PrettyPrint component once per page

## Example

<Canvas>
  <PrettyPrint initialize>{`<PrettyPrint initialize />`}</PrettyPrint>
</Canvas>

## Tag Type

Use the `tagType` attribute to set the tag to either `<pre>` or `<code>`

## Example Pre

<Canvas>
  <Story name="Pre" args={{ tagType: 'pre' }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Example Code

<Canvas>
  <Story name="Code" args={{ tagType: 'code' }}>
    {Template.bind({})}
  </Story>
</Canvas>
