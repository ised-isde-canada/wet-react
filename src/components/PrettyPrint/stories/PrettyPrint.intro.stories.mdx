# PrettyPrint

Provides syntax highlighting of source code snippets in an HTML page

## Example

<Canvas>
  <PrettyPrint>
    <p>for x in range (5):</p>
    <p>print("Hello There!")</p>
  </PrettyPrint>
</Canvas>
