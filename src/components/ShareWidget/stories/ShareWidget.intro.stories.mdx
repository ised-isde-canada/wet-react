# ShareWidget

Use to provide links that share content on various platforms (e.g., Twitter, Facebook, etc.)

## Example

<Canvas>
  <ShareWidget
    shareLinkText="Share this page"

    url="http://example.com"
    filter={[
      'Email',
      'Facebook',
      'Blogger',
      'LinkedIn',
      'Pinterest',
      'reddit',
      'tumblr',
      'WhatsApp',
      'Yahoo Mail',
      'Twitter',
    ]}
  />
</Canvas>
