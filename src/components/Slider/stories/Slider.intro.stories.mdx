# Slider

The `<input type="range" />` element enables the display of a slider control bar.

Because some browsers do not support this functionality natively, this polyfill emulates the same functionality using generic HTML and WAI-ARIA.

## Example

<Canvas>
  <Slider min={0} max={100} value={37} title="Example Slider" />
  <input
    type="range"
    min="0"
    max="100"
    step="1"
    title="Range: 0 to 100 in steps of 1"
    value="50"
  />
  <input type="range" min="-10" max="10" />
</Canvas>
