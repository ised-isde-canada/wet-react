# Tabs

## Tab List

on browsers with width>992px the tabs will default to a larger view mode

To handle this larger mode you must add a `Tabs.List` with a `Tabs.ListItem` for each tab

Each `Tabs.ListItem` should have similar values to its relevant `Tabs.PanelItem`:

<ul>
  <li>Same order</li>
  <li>Tabs.ListItem.href = Tabs.PanelItem.id</li>
  <li>Tabs.ListItem.active = Tabs.PanelItem.open</li>
</ul>

## Example

<Canvas>
  <Tabs

      <Tabs.List>

    />
  </Tabs>
