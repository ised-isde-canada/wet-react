# Tabs

Each `Tabs` component can have a unique custom id

# TabPanel

Each `TabPanel` **must** have a unique custom id, which is used for the mainPanel attribute in Tabs

## Example

<Canvas>
  <Tabs
    id="uniquevalue"
    mainPanel="text2"
    panels={[
      { title: 'tab one', content: <p>Tab Content</p>, id: 'text1' },
      { title: 'tab two', content: <p>Second Tab Content</p>, id: 'text2' },
      {
        title: 'tab three',
        content: <p>Third Tab Content</p>,
        id: 'text3',
      },
    ]}
  />
</Canvas>
