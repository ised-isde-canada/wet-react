# Tabs

Each `Tabs` component can have a unique custom id

# TabPanel

Each `TabPanel` component can have a unique custom id, used for the mainPanel attribute in Tabs

## Example

<Canvas>
  <Tabs
    id="uniquevalue"
    mainPanel="text2"
    panels={[
      { title: 'tab one', content: <p>I am a tab!</p>, id: 'text1' },
      { title: 'tab two', content: <p>I am also a tab!</p>, id: 'text2' },
      {
        title: 'tab three',
        content: (
          <p>
            Ladies and gentlemen, it is my pleasure to announce to you today,
            that we do indeed have... a third tab!
          </p>
        ),
        id: 'text3',
      },
    ]}
  />
</Canvas>
