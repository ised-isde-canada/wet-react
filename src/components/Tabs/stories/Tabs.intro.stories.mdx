# Tabs

Display images and other content in a dialog box, either individually or as part of a gallery.
Implements the<a href="https://www.w3.org/TR/wai-aria-practices/#dialog_modal" target="_blank"> WAI-ARIA Dialog (Modal) design pattern</a>

## Example

<Canvas>
  <Tabs
    id="uniquevalue"
    panels={[
      { title: 'tab one', content: <p>I am a tab!</p>, id: 'text1' },
      { title: 'tab two', content: <p>I am also a tab!</p>, id: 'text2' },
      {
        title: 'tab three',
        content: (
          <p>
            Ladies and gentlemen, it is my pleasure to announce to you today,
            that we do indeed have... a third tab!
          </p>
        ),
        id: 'text3',
      },
    ]}
  />
</Canvas>
