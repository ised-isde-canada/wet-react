# Tabs

Display images and other content in a dialog box, either individually or as part of a gallery. Implements the<a href="https://www.w3.org/TR/wai-aria-practices/#dialog_modal" target="_blank"> WAI-ARIA Dialog (Modal) design pattern</a>

## Example

<Canvas>
  <Tabs mainPanel="contact" id="hi">
    <TabRB eventKey="home" title="Home">
      <p>one</p>
    </TabRB>
    <TabRB eventKey="profile" title="Profile">
      <p>two</p>
    </TabRB>
    <TabRB eventKey="contact" title="Contact">
      <p>tre</p>
    </TabRB>
  </Tabs>
</Canvas>
