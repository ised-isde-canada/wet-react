# Tabs

Each `Tabs` component can have a `mainPanel` attribute, which contains the id of the tab to have open by default

## Example

<Canvas>
  <Tabs
    id="unique1"
    mainPanel="text2"
    panels={[
      { title: 'Tab 1', content: <p>Content 1</p>, id: 'text1' },
      { title: 'Tab 2', content: <p>Content 2</p>, id: 'text2' },
      {
        title: 'Tab 3',
        content: <p>Content 3</p>,
        id: 'text3',
      },
    ]}
  />
</Canvas>

If `mainPanel` is not specified, it will default to the first tab

If `mainPanel` is an id that does not exist, no panels will be open by default. (this is an intentional feature)
