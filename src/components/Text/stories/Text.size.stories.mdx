# Text

## Text sizing

You can override the global default font-size of `16px`, with a line-height of `1.428`.

**Please note:**

- `h1-h6` modify the size of the text. This is done only through styling. It will not affect page structure.
- `small` is used to define small text, for example side-comments, disclaimers and "fine" print, including copyright and legal text.
- `lead` Make a paragraph stand out by adding setting this to true
